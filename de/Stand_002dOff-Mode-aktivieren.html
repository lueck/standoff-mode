2.1 Stand-Off Mode aktivieren

Tast.MenüKommando
standoff-mode

Nach dem Öffnen einer Datei und ihrer Darstellung in einem Buffer kann für diesen Buffer der Stand-Off Mode aktiviert werden durch Eingabe von M-x standoff-mode RET, also kurzes Drücken von ESC, gefolgt von einem kurzen Drücken von x, gefolgt von der Eingabe von standoff-mode im dann aktiven Minibuffer und anschließendes Betätigen der Eingabe-Taste.

Im Stand-Off Mode ist der Buffer read-only, ein Editieren des Textes ist also nicht möglich. In der Modeline wird (Stand-Off) als aktiver Modus angezeigt. Zudem erscheint im Hauptmenü der Punkt ‘Stand-Off’.

Bei Aktivierung von Stand-Off Mode werden bereits früher erfasste Annotationen geladen. See Bereits früher erfasstes Markup laden.
