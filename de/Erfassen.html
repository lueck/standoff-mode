3.1 Erfassen

Tast.MenüKommando
m‘Create new markup element’standoff-markup-region

Mit diesem Kommando wird ein neues Markup-Element erfasst. Es erstreckt sich über die active Region, also den ausgewählten Text-Ausschnitt. Im Minibuffer wird nach dem Typ des zu erfassenden Elements gefragt. Durch Drücken von TAB (=Tabulator) erhält man eine Auswahl bzw. eine begonnene Eingabe wird vervollständigt. Das neu erstellte Element wird visuell hervorgehoben (highlight).
