3.4 Buffer bzw. Datei schließen und Emacs beenden

Tast.MenüKommando
C-x k‘File --> Close’kill-buffer
C-x C-c‘File --> Quit’save-buffers-kill-terminal

Mit diesen Kommandos schließen Sie die Datei bzw. Emacs.

Denken Sie daran, vorher Ihre Arbeiten zu sichern. See Speichern.
